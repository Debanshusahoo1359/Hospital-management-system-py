# 🏥 Hospital Management System

A simple **Hospital Management System** developed in **Python**. This project is a menu-driven application that helps manage hospital patient records efficiently. It allows users to add, view, search, update, and delete patient records through a simple command-line interface.

---

## ✨ Features

- ➕ Add New Patient
- 📋 View All Patients
- 🔍 Search Patient by ID
- ✏️ Update Patient Details
- ❌ Delete Patient Record
- 🆔 Auto-generated Patient ID
- 👨‍⚕️ Store Doctor Name
- 🩺 Store Disease Information
- 📌 Patient Admission Status
